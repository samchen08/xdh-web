# xdh-back-top 返回顶部组件

当页面内容冗长，需要快捷返回顶部时使用，一般放置在页面右下角位置。

## 用法

```
  <xdh-back-top></xdh-back-top>
```

## 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|----|----|----|----|----|
| height | 页面滚动高度达到该值时才显示组件 | Number | - | 400 |
| bottom | 组件距离底部的距离 | Number | - | 30 |
| right | 组件距离右部的距离 | Number | - | 30 |
| duration | 滚动动画持续时间，单位 毫秒 | Number | - | 1000 |

## 插槽

| name | 说明 |
|-----|-----|
| - | 自定义内容 |


## 方法

无
