# xdh-nav-tabs 导航tabs


## 用法
```
    <xdh-nav-tabs label="首页" name="/home" :closable="false" store="system"></xdh-nav-tabs>
```

## 属性
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|----|----|----|----|----|
| label | 显示文字 | String | - | - |
| name | 选项卡名称，通常设为路由的 path | String | - | - |
| closable | 是否能关闭 | Boolean | - | true |
| store | store的模块名称 | String | - | xdhNavTabs |
